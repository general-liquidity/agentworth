---
name: build-emergency-buffer
description: Strengthen the economic pillar by building toward a 3-month emergency buffer.
---

When the operator's weakest pillar is **economic** and the emergency buffer is
under 3 months of essentials:

1. Compute the gap: target = 3 × monthly essentials; gap = target − liquid savings.
2. Propose a small, automatic recurring transfer that fits the monthly surplus —
   never one that pushes essentials into deficit.
3. Frame it as one reversible step; if anxiety is high, start with a token amount
   and build the habit before the size.
4. Create the transfer as a `pay`/mandate proposal — it still passes the gate.
Never recommend high-cost credit to "free up" cash for saving.
